# MAS Singapore Data

**Source:** Monetary Authority of Singapore
**API Portal:** https://eservices.mas.gov.sg/api/
**Docs:** https://eservices.mas.gov.sg/apidocs/

## Data Available
- Interest rates (SORA, SOR, SIBOR)
- Exchange rates (SGD)
- Money supply
- Banking statistics
- Securities market data
- Fund management statistics
- Domestic interest rates

## Signal Value
Singapore monetary data - key Asian financial hub.

## Implementation Notes
- REST API with JSON responses
- No authentication required for most endpoints
- Data organized by category
- Daily/monthly frequencies
- Key series: SORA, exchange rates, money supply
- Well-documented API
